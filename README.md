# FOOD-PWA 🍔

This project is an **E-commerce Progressive Web App (PWA)**.

##  Features
- Fast and responsive UI
- Installable (Add to Home Screen)
- Service Worker support
- Offline capability

## 🛠️Technologies Used
- HTML
- CSS
- JavaScript
- PWA (Manifest + Service Worker)
- Python
